# rania-fds
A Fall Detection System (FDS) for RANIA

## Development

### Requirements

rania-fds requires the following software & libraries:

* Python 3.11
* pip:
  * pipenv
  * rplidar

### Installing Dependencies

After cloning, first install `pipenv` via `pip` and install dependencies:
```
pip install pipenv
cd ./rania-fds
pipenv install
```
